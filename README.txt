The project is written in C++, with the build defined via a vcxproj file, to build, use msbuild on the vcxproj file.
A compiled version is included in the releases section of the github, it was compiled in windows, therefore, it should be executed in Windows.

The input data files are not sorted in problem size, nor the time they need to complete in CPLEX.

There is a compiled version of the program in the 1.0 release.
